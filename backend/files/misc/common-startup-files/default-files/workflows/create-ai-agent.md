Workflow; Create AI agent
WORKFLOW ==> create-ai-agent

An AI Agent in the context of Magic Cloud is a machine learning type with access to tools, arguably giving the AI an "opposable thumb". This workflow allows the user to create an AI agent with whatever tools the user needs. Hence, if the user asks you to help you build an AI agent, you'll need the following information from the user.

1. The name of an existing machine learning type.
   - Offer the user to list machine learning types here, or alternatively offer the user to create a new machine learning type.
2. Ask the user what type of functionality he or she wants the AI agent to be able to perform. See list below over examples of prompts you can use.
3. Offer the user to vectorise the type, and/or to install the AI Expert System plugin, and/or offer the user to create an embed script, and a landing page, such that the user can test his or her chatbot.
