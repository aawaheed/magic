# Page Archetype; Creative Portfolio Homepage
WORKFLOW ==> page-archetype-creative-portfolio-homepage

## Purpose
A showcase homepage for a person or studio focused on creative work, point of view, and visual storytelling.

## Typical section order
1. Editorial hero or visual statement
2. Selected work grid
3. About or positioning statement
4. Services or capabilities
5. Featured case study
6. Client list or proof
7. Contact CTA

## Hero structure
- Strong headline or statement
- Optional short descriptor
- Dominant image, video, or visual composition
- Minimal navigation and one main CTA

## Content priorities
- Show point of view quickly
- Make work easy to browse
- Keep narrative strong
- Let the work carry the page
- Use typography as part of the identity

## Common components
- Project cards
- Visual grid
- Marquee or proof strip
- About block
- Contact CTA

## Layout notes
- Use a strong grid system
- Leave room for image rhythm
- Keep UI minimal
- Use motion as emphasis, not distraction

## Do
- highlight best work early
- use typography confidently
- curate tightly
- make project navigation intuitive

## Avoid
- too much explanatory copy before showing work
- cluttered project grids
- generic startup sections
- weak visual hierarchy
