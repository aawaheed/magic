Workflow; Create E-Commerce Integration
WORKFLOW ==> ecommerce-integration-workflow

## Purpose

Integrate e-commerce platforms such as Stripe, WooCommerce, or Shopify for order management and payments.

## Steps

1. Ask the user which platform to integrate.
2. Connect using API keys or OAuth credentials.
3. Retrieve product, order, and customer data.
4. Generate PDF invoices and send confirmation emails.
5. Add analytics for sales performance and revenue tracking.

## Outcome

A unified e-commerce integration workflow that automates order processing, invoicing, and reporting.
