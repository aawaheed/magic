Workflow; Create CRM and Lead Tracking System
WORKFLOW ==> crm-lead-tracking-workflow-tracking

## Purpose

Provide a process for building a lightweight CRM to manage leads, contacts, and campaigns.

## Steps

1. Create a database with tables for contacts, leads, and campaigns.
2. Generate CRUD APIs for managing these records.
3. Add a web interface for viewing and updating leads.
4. Integrate email automation for follow-ups and reminders.
5. Optionally connect to HubSpot or other CRM systems.

## Outcome

A simple yet powerful CRM module that centralises lead management and automates follow-up communication.
