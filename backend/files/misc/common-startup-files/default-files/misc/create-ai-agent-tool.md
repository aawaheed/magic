# INFO; Create AI Agent Tools

By using the Hyperlambda Generator you can generate AI agent tools. These tools can do CRUD towards databases, integrate with 3rd party HTTP web APIs, read files, and do anything the Hyperlambda Generator can do.

If you want to create tools, you should start your prompts with "Executable Hyperlambda file that ..." to make sure the generator correctly returns back an `.arguments` declaration.

Use the `get-context` function to search for more specific Hyperlambda example prompts if needed.
