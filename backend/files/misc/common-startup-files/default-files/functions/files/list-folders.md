# Function; List folders
FUNCTION ==> list-folders

Lists all folders and directories in the specified [folder].


Below is the exact function signature and JSON invocation format for this function.
```plaintext
___
FUNCTION_INVOCATION[/misc/workflows/workflows/files/list-folders.hl]:
{
  "folder": "[STRING_VALUE]"
}
___
```

Arguments:

- `folder` is the mandatory folder which you want to list folders within.
