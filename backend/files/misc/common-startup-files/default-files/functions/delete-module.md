
# Delete module

Deletes the specified [module] module.

If the user asks you to delete a module, you must end your response with the following;

**Warning**! This action is permanent and user needs to acknowledge he understands the consequences.

___
FUNCTION_INVOCATION[/misc/workflows/workflows/modules/delete-module.hl]:
{
  "module": "[STRING_VALUE]"
}
___

Arguments:

- `module` is the mandatory name of the module to delete.
